在windows10中安装Mac OS X 10.12.5

在windows10中安装Mac OS X 10.12.5

基本环境及工具

Mac OS X 10.12.5 下载文件 提取码：vbxi

unlocker208下载文件 提取码：1lax

vmware在360软件管家中下载最新版的即可，此处用的vmware版本为12.5.7

安装工具的步骤

vmware：

安装时一直点击下一步，顺着步骤往下走，最后在网上百度个序列号就能用。

unlocker：

unlocker208是vmware虚拟机中安装苹果系统的破解补丁。我试过unlocker203、 207版本。vmware均无法识别mac系统。如果你使用208，但是还不可以，可以试试看重启，或许能解决识别不了的问题:)

下载完unlocker后，进行解压。

安装系统

  • 按“win+r”按键，输入“services.msc”，确定后将有关于vmware的五项服务右击选择“停止”全部停掉。

  • 然后去unlocker208的文件中，找到win-install.cmd文件，以管理员身份运行。如果不进行这一步的话，在vmware中新建虚拟机时“选择客户机操作系统”中不会出现Apple Mac OS X这个选项。

    CB%AO5AIEIJYE7G~Z%E`2.png)

  • 打开安装好的vmware，在主页中，选择“创建新的虚拟机”

  • 类型选择为“典型”

  • 点击下一步，选择“安装程序光盘映像文件”并且选择你下载的mac系统的文件路径。

    将文件类型改成全部，才能够看到下载的系统文件。

    此时，下方会提示“无法检测此光盘映像中的操作系统”等内容，不要理会。

  • 点击下一步，此时客户机操作系统中会出现“Apple Mac OS X”的选项，选择版本为macOS 10.12版本。若此时没有出现“Apple Mac OS X”的选项，很有可能是unlocker的安装有问题，请返回这一步检查。

  • 点击下一步，定义虚拟机的名称，并且选择所在的位置。一般会默认装在C盘，但考虑到C盘为系统盘，尽量不要装在C盘

  • 点击下一步，定义磁盘大小为40GB(可以根据需要自行调节)，选择讲虚拟磁盘拆分成多个文件。

  • 点击下一步，可以选择“自定义硬件”，可以调节内存等内容，根据需要自行调节。

  • 点击完成，则安装完成。

  • 完成后，点击开启此虚拟机将会出现“不可恢复的错误”

  • 解决方法为：

    打开安装系统的目录中，找到“macOS 10.12.vmx”文件，并右击使用记事本打开，找到其中的语句”smc.present=”True””并且在下方添加语句“smc.version=0”，点击保存后，重新开启虚拟机，会发现问题已经解决。

  • 选择语言，并且同意条款之后，出现“选择要安装到的磁盘”，这时可能出现，无法继续的情况。此时，点击“实用窗口”中的“磁盘工具”选项。

  • 点击vmware选项，并且选择“抹掉”。这时回来原来的界面，发现多了一个选项，点击就可以继续下一步了

  • 配置了用户名等基本信息后，安装成功！
